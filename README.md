# Billing-Machine-Project
## 📌 Description
This project is a console-based Billing System developed using Python. It allows users to select items from a predefined menu, input quantities, and automatically calculates the total bill.

The system simulates a real-world restaurant billing process by handling multiple orders, validating item availability, and generating a final bill based on user input.

## 🚀 Features
- Menu-based item selection
- Quantity input for each item
- Automatic total bill calculation
- Handles multiple orders in a single session
- Validates item availability
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Python

## 📚 Purpose
This project was built to strengthen programming fundamentals such as loops, conditionals, dictionaries, and user input handling while simulating a real-world billing system.
